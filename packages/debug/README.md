# nestjs-arktype-debug

デバッグ・実験用スクリプトのワークスペース

## 概要

このワークスペースには、本体の実装とは関係のない、デバッグや調査用のスクリプトが含まれています。

## スクリプト

### debug-nullable.ts
Nullable型のスキーマ変換を確認するスクリプト

```bash
pnpm debug-nullable
```

### zod-test.ts
nestjs-zodの`_OPENAPI_METADATA_FACTORY`パターンを調査するスクリプト

```bash
pnpm zod-test
```

### simple-test.ts
最もシンプルなArkType DTOの動作確認スクリプト

```bash
pnpm simple-test
```
